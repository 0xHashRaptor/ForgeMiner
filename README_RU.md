## ⚒️ Обзор

**ForgeMiner** — высокопроизводительный полностью нативный майнер для видеокарт NVIDIA. Он работает с видеокартой напрямую через CUDA Driver API — **без Python, без WSL, без лишних рантаймов** — поэтому запускается мгновенно и потребляет минимум ресурсов даже на слабых ригах. **Первая поддерживаемая монета — Pearl (PRL)** (Proof-of-Useful-Work); скоро будут и другие.

Ядро под Pearl — вручную оптимизированный движок на тензорных ядрах, с отдельной сборкой под каждое поколение карт, чтобы каждый GPU работал на максимуме.

> ForgeMiner — закрытый (исходники не публикуются). Релизы выходят в нашем [Telegram-канале](https://t.me/ForgeMiner).

## ✨ Особенности майнера

- 🚀 **Ядра под архитектуру** — отдельное ядро под каждое поколение (Turing / Ampere / Ada / Blackwell), выбирается автоматически при запуске. Лидер по скорости на RTX 40-й серии.
- 🆕 **v1.0.7** — **быстрее на RTX 50-й серии (Blackwell)**: перетюнен этап hash-fold под планировщик 50-й серии, **~+4–5%** на тех же клоках — наравне с самыми быстрыми майнерами. RTX 30/40 без изменений.
- 🆕 **v1.0.4** — выбор карт для майнинга (`--gpu 0,1,2,6`) и **разный разгон на каждую карту**; фикс стабильности дашборда на долгих прогонах.
- 🆕 **v1.0.3** — RTX **20-я серия (Turing) теперь майнит на Stratum-пулах** (HeroMiners / LuckyPool / Kryptex), а не только AlphaPool; **+4%** на RTX 30 / 40 / 50.
- ⚡ **Нативный и лёгкий** — прямой CUDA Driver API, почти нулевая нагрузка на CPU (blocking-sync), отлично работает на слабых и многокарточных ригах.
- 🔒 **Самодостаточный и защищённый** — единый бинарь, всё вшито и зашифровано; никаких отдельных файлов-ядер, которые нужно таскать или которые можно «утащить».
- 🔥 **Встроенный разгон** — лок частот, оффсеты ядра/памяти и лимит мощности прямо из майнера (сторонние OC-утилиты не нужны).
- 🌐 **Несколько пулов и диалектов** — из коробки работает со стандартными Pearl Stratum пулами (BaikalMine, HeroMiners, LuckyPool, Kryptex) и с диалектом AlphaPool.
- 🖥️ **Наглядная панель** — по каждой карте: хешрейт, принятые/устаревшие/отклонённые шары, эффективность (GH/W), температуры, частоты, вентиляторы и мощность.
- 🛠️ **Готов к HiveOS** — ставится в кастомный слот HiveOS.
- ♻️ **Устойчивый** — автопереподключение и переключение между пулами; конвейер майнинга не встаёт.

## 🚀 Быстрый старт

### 🪟 Windows
1. Скачайте и распакуйте релиз для **Windows**.
2. Откройте `.bat` своего пула (`Baikal.bat`, `HeroMiners.bat`, `LuckyPool.bat`, `AlphaPool.bat`) в текстовом редакторе и впишите свой **кошелёк** и **имя воркера**.
3. Запустите `.bat` двойным кликом. *(Запустите от имени Администратора, если хотите, чтобы применился встроенный разгон.)*

### 🐧 Linux
```bash
chmod +x forge
FORGE_POOL=ru.pearl.herominers.com:1200 \
FORGE_WALLET=ВАШ_КОШЕЛЁК FORGE_WORKER=rig01 \
FORGE_PROTO=stratum ./forge
```
…или используйте вложенный `start.sh`, отредактировав кошелёк.

### ⛏️ HiveOS
Создай полётный лист **Custom miner** с такими полями:

| Поле | Значение |
|---|---|
| **Имя майнера** | `ForgeMiner` *(подставится из URL автоматически)* |
| **Установочный URL** | `https://github.com/0xHashRaptor/ForgeMiner/releases/download/v1.0.7/ForgeMiner-1.0.7.tar.gz` |
| **Хэш алгоритм** | *(оставить пустым)* |
| **Кошелёк и воркер шаблона** | `%WAL%.%WORKER_NAME%` *(если привязан Pearl-кошелёк)* — или впиши явно `ВАШ_КОШЕЛЁК.%WORKER_NAME%` |
| **Адрес пула** | `pearl.baikalmine.com:2010` *(BaikalMine, комиссия 0.5%)* · `ru.pearl.herominers.com:1200` *(HeroMiners)* · `45.151.62.119:3361` *(LuckyPool)* · `prl-ru.kryptex.network:7048` *(Kryptex)* |
| **Пароль** | `x` |
| **Доп. параметры конфигурации** | *пусто для Stratum* · `FORGE_PROTO=alpha` для AlphaPool · разгон, напр. `FORGE_CCLK=2505` |

Применить → в дашборде появятся хешрейт, температуры, кулеры и шары по картам. **Один** майнер на риг.

## ⚙️ Параметры майнера

Параметры можно задавать как **флаги командной строки** *(`--флаг значение`)* **или** как **переменные окружения** *(`FORGE_ФЛАГ`)* — удобно для HiveOS и `.bat`-файлов.

| Флаг | Переменная | Описание |
|------|------------|----------|
| `--algorithm` | `FORGE_ALGO` | Алгоритм (`pearl`). |
| `--pool` | `FORGE_POOL` | Адрес пула в виде `host:port`. |
| `--wallet` | `FORGE_WALLET` | Кошелёк для выплат. |
| `--worker` | `FORGE_WORKER` | Имя воркера/рига на пуле. |
| `--password` | `FORGE_PASS` | Пароль пула (обычно `x`). |
| `--proto` | `FORGE_PROTO` | Диалект пула: `stratum` (HeroMiners, LuckyPool, Kryptex) или `alpha` (AlphaPool). |
| `--gpu` | `FORGE_GPU` | Майнить только на этих индексах карт, напр. `--gpu 0,1,2,6` (по умолчанию — все). Индексы как в `nvidia-smi`. |
| — | `FORGE_LOWVRAM` | Режим экономии видеопамяти для карт на 8 ГБ (примерно вдвое меньше VRAM): `1` — включить, `0` — выключить. По умолчанию определяется автоматически. |

### 🔥 Разгон
> 💡 **Совет:** ForgeMiner **упирается в частоту ядра и почти не зависит от памяти** — для максимального хешрейта ставьте **высокое ядро**; память может оставаться низкой. Разгон требует **root** (Linux/HiveOS) или **Администратора** (Windows).
>
> 🆕 **Разный разгон на карту:** каждый флаг принимает одно значение (на все карты) **или список через запятую**, который соответствует `--gpu` — для ригов со смешанными картами:
> `--gpu 0,1,2,6 --coff 300,250,300,200 --plimit 280,280,300,260`

| Флаг | Переменная | Описание |
|------|------------|----------|
| `--cclk` | `FORGE_CCLK` | Лок частоты ядра (МГц). |
| `--coff` | `FORGE_COFF` | Оффсет частоты ядра (МГц, `+`/`-`). |
| `--mclk` | `FORGE_MCLK` | Лок частоты памяти (МГц). |
| `--moff` | `FORGE_MOFF` | Оффсет частоты памяти (МГц, `+`/`-`). |
| `--plimit` | `FORGE_PLIMIT` | Лимит мощности (Вт). |
